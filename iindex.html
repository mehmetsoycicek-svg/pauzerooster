Configureer GitHub om wijzigingen voor iedereen op te slaan.

Stap-voor-stap instructie:
  1. Ga naar GitHub Tokens
  2. Klik op "Generate new token (classic)"
  3. Geef een naam bijv. "Pauzerooster"
  4. Selecteer alleen "repo" scope (vinkje aan)
  5. Klik "Generate token" onderaan
  6. Kopieer het token (dit zie je maar 1x!)
